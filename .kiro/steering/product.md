# Learning Together Platform (Pactly)

## Product Overview

A web-based learning platform that helps people learn better together through small accountability pods (2-4 people), structured sessions, and an AI Session Coach.

## Core Features

- **Pod-based Learning**: Small groups for accountability and peer support
- **AI Session Coach**: Generates structured agendas, tracks progress, creates quizzes
- **Sprint Templates**: Pre-built learning tracks (Gym 3×/week, .NET Prompting)
- **Progress Tracking**: Daily check-ins with RPE scoring, wins/tweaks reflection
- **Video Integration**: Embedded Jitsi Meet and external video platform support
- **Smart Matching**: Algorithm-based pod formation using compatibility signals
- **Gamification**: Streak tracking, proof of effort submissions
- **Payment System**: Optional refundable deposits for motivation

## Key User Flows

1. **Onboarding**: Goal creation → Sprint template selection → Time slot preferences → If-then planning
2. **Matching**: Algorithm matches users into pods based on preferences and availability
3. **Sessions**: Scheduled video sessions with AI-generated agendas and materials
4. **Check-ins**: Daily progress tracking with proof of effort
5. **Progress**: Streak tracking, session attendance, goal completion

## Domain Concepts

- **Sprint**: Time-boxed learning period with specific goals
- **Pod**: Small learning group (2-4 members) with shared sprint type
- **Session**: Scheduled video meeting with structured agenda
- **Check-in**: Daily progress report with reflection and proof
- **Coach Doc**: AI-generated session materials (agenda, notes, quiz)
